# Task Management App 📋

Une application complète de gestion des tâches créée avec HTML, CSS et JavaScript.

## ✨ Fonctionnalités

- ✅ **Ajouter des tâches** - Créez rapidement de nouvelles tâches
- 🎯 **Marquer comme complétée** - Cochez les tâches terminées
- ✏️ **Éditer les tâches** - Modifiez le texte d'une tâche
- 🗑️ **Supprimer les tâches** - Supprimez une tâche spécifique
- 🔍 **Filtrer les tâches** - Filtrez par Tous, En cours, ou Complétées
- 📊 **Statistiques en temps réel** - Suivi du nombre de tâches
- 💾 **Sauvegarde locale** - Les tâches sont sauvegardées automatiquement (LocalStorage)
- 📱 **Responsive design** - Fonctionne sur tous les appareils

## 🚀 Comment utiliser

1. Ouvrez `index.html` dans votre navigateur
2. Tapez une tâche dans le champ d'entrée
3. Cliquez sur "Ajouter" ou appuyez sur Entrée
4. Utilisez les contrôles de chaque tâche :
   - ✓ Case à cocher pour marquer comme complétée
   - ✏️ Bouton édition pour modifier
   - 🗑️ Bouton suppression pour supprimer
5. Utilisez les filtres pour afficher les tâches souhaitées

## ⌨️ Raccourcis clavier

- **Entrée** - Ajouter une nouvelle tâche
- **Click** - Cocher/décocher une tâche

## 💾 Stockage des données

Les tâches sont sauvegardées automatiquement dans le **LocalStorage** de votre navigateur. Cela signifie que vos tâches persisteront même après le rechargement de la page.
